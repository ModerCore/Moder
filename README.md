<h1 align="center">
    Moder
</h1>

Moder is a visual editor for HOI4 (Hearts of Iron IV)  that simplifies Mod development.

## Runtime environment requirements

- Windows 10 1809 - Build 1776 as well as Versions above

## Funtions

- state File Editing
- Character Editor (WIP)
